# Formação Python Fundamentals - DIO 🐍

Este repositório contém todos os artefatos, desafios de código e projetos práticos que desenvolvi durante a trilha **Python Fundamentals** na [DIO (Digital Innovation One)](https://www.dio.me/).

O objetivo desta formação é consolidar o conhecimento desde a sintaxe básica até conceitos avançados de Programação Orientada a Objetos e manipulação de dados.

## 📚 Conteúdo do Repositório

O conteúdo está organizado de acordo com os módulos do curso:

* **Fundamentos de Python:** Variáveis, operadores, estruturas condicionais e de repetição.
* **Estruturas de Dados:** Manipulação de listas, tuplas, conjuntos e dicionários.
* **Programação Orientada a Objetos (POO):** Classes, herança, encapsulamento, polimorfismo e interfaces.
* **Python Avançado:** Decoradores, iteradores, geradores e manipulação de arquivos (CSV, TXT) e datas.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Python 3.x
* **IDE:** Visual Studio Code (VS Code)
* **Versionamento:** Git e GitHub

## 📂 Estrutura de Pastas
- `/desafios-de-codigo`: Resoluções de lógica e algoritmos propostos pela plataforma.
- `/projetos`: Projetos práticos que compõem o portfólio.
- `/exercicios`: Scripts de fixação de sintaxe e conceitos.
